# Data Folder

This folder stores project inputs and source tables.

- top-level CSV/TXT/XLSX files are manuscript tables, genotype matrices, and coordinate inputs
- `metadata/` contains derived lookup tables and crosswalk files
- `P_globosa_map/` contains local shapefiles used for the custom *P. globosa* map overlays
- `STRUCTURE_named/` and `STRUCTURE_out/` contain STRUCTURE-related exports
- `natural_earth_10m/` and `natural_earth_50m/` contain bundled Natural Earth reference layers
